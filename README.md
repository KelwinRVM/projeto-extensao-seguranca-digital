# CyberSafe Community

Aplicativo web educativo desenvolvido em Python e Streamlit como projeto de extensão do curso de Segurança da Informação.

## Objetivo

Conscientizar a comunidade sobre segurança digital, prevenção contra golpes virtuais, proteção de dados pessoais, criação de senhas seguras e uso seguro da internet.

## Funcionalidades

- Checklist de golpes digitais
- Teste educativo de senha
- Quiz de segurança digital
- Dicas rápidas de proteção
- Formulário de feedback da comunidade

## Tecnologias utilizadas

- Python
- Streamlit
- GitHub
- Streamlit Community Cloud
